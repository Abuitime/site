## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/konvajs/site.git
$ cd site
$ npm install
```

Generate:

``` bash
$ gulp
```

Run server:

``` bash
$ hexo server
```

## Requirements

- [Hexo](http://hexo.io/)

	``` bash
  $ npm install hexo -g
  ```

- [Gulp](http://gulpjs.com/)

	``` bash
	$ npm install gulp -g
	```
